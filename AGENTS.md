# Codex / OpenAI Agent Instructions

This repository contains Warren Buffett's shareholder letters (1977-2025), Berkshire Hathaway portfolio holdings data, and curated investment quotes.

## Quick Reference

When asked about Buffett's investment wisdom:
1. Read `data/letters/{year}.md` for annual summaries
2. Search `data/quotes.json` for quotes (bilingual EN/ZH)
3. Check `data/holdings/summary.json` for portfolio data
4. Read `docs/investment_philosophy.md` for core principles
5. Read `docs/portfolio_evolution.md` for historical analysis

## File Structure
- `data/letters/` — Annual shareholder letter summaries (Markdown)
- `data/quotes.json` — 50+ curated quotes with year, category, EN/ZH
- `data/holdings/` — Portfolio holdings data (JSON)
- `docs/` — Deep analysis reports (Markdown)

## Supported Queries
- Annual letter lookups by year
- Quote searches by theme/category
- Portfolio holdings history
- Investment philosophy questions
- Comparative analysis across years
