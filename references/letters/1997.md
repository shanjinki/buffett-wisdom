# 1997 年致股东信

## 📅 年度概要

1997年伯克希尔股价首次突破5万美元/股。科技股泡沫开始酝酿，巴菲特坚持不买科技股，虽然面临巨大的市场压力。

- 标普500上涨33%
- 伯克希尔净值增长35%
- 大幅增持可口可乐和吉列
- 拒绝投资互联网公司

## 💡 核心投资观点

### 1. 不懂不投
- 坚决不投资看不懂的科技股
- 宁可错过，不可做错
- 承认自己的能力圈有限

### 2. 品牌的力量
- 可口可乐、吉列等品牌有持久的竞争优势
- 消费垄断是最好投资标的
- 品牌资产是最难复制的

### 3. 通货膨胀与投资
- 通胀侵蚀购买力
- 只有企业才能跑赢通胀
- 股票是抵御通胀的工具

### 4. 企业收购标准
- 六条核心标准不变：
  1. 大型收购（税后利润至少5000万美元）
  2. 持续盈利能力
  3. 良好的股东权益回报率
  4. 管理层已证明的能力
  5. 简单的业务模式
  6. 合理的价格

### 5. 对市场的看法
- 市场先生仍然情绪化
- 牛市中保持理性更难
- 估值越高，越要谨慎
