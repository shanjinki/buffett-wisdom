# 1995 年致股东信

## 📅 年度概要

1995 年伯克希尔表现优异，市场强劲上涨。

- 伯克希尔每股账面价值变化：+43.1%
- S&P 500（含股息）：+37.6%
- 超额收益：+5.5 个百分点
- 科技股开始崛起
