# 1996 年致股东信

## 📅 年度概要

1996 年伯克希尔表现稳健，市场继续上涨。

- 伯克希尔每股账面价值变化：+31.8%
- S&P 500（含股息）：+23.0%
- 超额收益：+8.8 个百分点
- 发行 B 股

## 💡 核心投资观点

### 1. 发行 B 股
- 为小股东提供 B 股
- 降低投资门槛
- 让更多投资者受益

### 2. 市场估值
- 市场估值偏高
- 警惕非理性繁荣
- 保持理性

### 3. 投资策略
- 坚持价值投资
- 不追逐热点
- 长期视角
