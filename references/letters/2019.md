# 2019 年致股东信

## 📅 年度概要

2019年伯克希尔的股票投资组合表现强劲，标普500大幅上涨。巴菲特继续执行大规模回购计划。

- 标普500全年上涨31%
- 伯克希尔净值增长11%
- 大规模回购公司股票
- 持续增持苹果

## 💡 核心投资观点

### 1. 回购的哲学
- 回购是提高股东价值的最佳方式之一
- 只在价格低于内在价值时回购
- 批评滥用回购的公司

### 2. 10年赌约的结果
- 详细回顾与对冲基金Protégé的十年赌约
- 标普500指数基金大幅跑赢对冲基金
- 强调低费用指数基金的长期优势

### 3. 并购的挑战
- 承认好交易越来越难找
- 市场估值偏高
- 需要保持耐心

### 4. 收购BNSF的10年回顾
- 铁路投资是成功的
- 基础设施的长期价值
- 但某些方面低于预期

### 5. 对投资的终极定义
- 投资是为未来购买力所做的部署
- 第一条规则：永远不要亏钱
- 第二条规则：永远不要忘记第一条
