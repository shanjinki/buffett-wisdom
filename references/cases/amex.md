# 美国运通投资案例

## 投资概况

| 项目 | 详情 |
|------|------|
| **首次投资** | 1964 |
| **持有年限** | 60 年 + |
| **性质** | 经典困境反转案例 |
| **当前持仓** | 约 2 亿股 |

## 投资背景

1963 年，美国运通遭遇"色拉油丑闻"：
- 子公司参与欺诈行为
- 面临巨额索赔
- 股价下跌 50%
- 市场担心公司倒闭

## 投资逻辑

### 1. 深度调查
- 巴菲特亲自调查
- 发现核心业务未受影响
- 品牌声誉仍然强大
- 问题出在子公司，非核心业务

### 2. 护城河分析
- **品牌优势**：全球知名的高端品牌
- **网络效应**：商户 + 用户双边网络
- **转换成本**：高，用户习惯难以改变
- **定价权**：可以收取高额费用

### 3. 投资规模
- 巴菲特将合伙公司 40% 资金投入
- 显示高度确信
- 适度集中投资

## 投资结果

- 丑闻最终解决，损失可控
- 股价大幅反弹
- 成为长期核心持仓
- 60 年持有期带来巨大回报

## 投资教训

1. **深度调查**：亲自验证，不轻信市场传言
2. **区分问题**：暂时性问题 vs 永久性损伤
3. **适度集中**：高度确信时可以加大投资
4. **长期持有**：优秀企业值得长期持有

## 巴菲特语录

> "当别人恐慌时，我们要保持理性。"

> "深入分析可以发现市场过度反应创造的机会。"

## 投资启示

1. **危机投资**：需要深入分析问题的本质
2. **品牌力量**：强大品牌可以度过危机
3. **逆向思维**：别人恐惧时贪婪
